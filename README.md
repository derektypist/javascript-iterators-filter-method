# JavaScript Iterators filter() Method

Applying the `filter()` method to a JavaScript Iterator.  The `filter()` method returns a new iterator containing elements that satisfy a filter function.  For example, alphabetical characters.

## UX

**Getting Started**

In the text field. enter text (e.g. Hello World).  You will see the text entered as well as the letters displayed (in lower case).  For example, `Hello World` has the letters `helloworld`.

**User Stories**

- As a user, I expect to see what text I have entered
- As a user, I expect to see what characters satisfy the entered text
- As a user, if I click on the `Clear` button, I expect the information to be cleared

## Features

Allows the user to enter text in order to see the letters of the alphabet that satisfy the text entered.

## Technologies

Uses HTML5, CSS3, Bootstrap 5.3.2 and jQuery 3.7.1.

## Testing

To preview, use the command `python3 -m http.server` and press `Enter`.  Check that all user stories have been met.
